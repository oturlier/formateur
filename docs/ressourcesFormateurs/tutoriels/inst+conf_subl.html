Sublime Text

Sublime Text est un éditeur de texte puissant, beau & efficace! C’est celui que je préfère!

Malgré l’efficacité de Sublime Text , cet éditeur ne propose pas de panneau de pré-visualisation du texte comme d’autres éditeurs que nous installerons par la suite : Atom, Visual code ou AsciidocFX

Nous pourrons pallier à ça en installant des plugins dans les navigateurs internet Firefox ou Google Chrome

Sublime Text n’est pas gratuit! 80 $ pour un usage personnel …​ mais dans la grande magnificence de l’auteur, pour pouvez continuer à l’utiliser gratuitement si vous supportez de fermer le pop-up qui vous demande régulièrement de passer à la caisse …​

Installation

Procédure : installation de Sublime Text

  1. se rendre sur le site web de Sublime Text (clic sur le lien précédent ou sur l’image pour ouvrir un autre onglet).

  1. cliquez sur Download for Windows pour télécharger la dernière version adaptée

  2. double-cliquez pour installer en suivant les instructions

Bonnes pratiques / Configuration

Ci-dessous quelques infos utiles pour éditer des ressources au format Asciidoc.

Mise en forme du texte “englobante”

  • après édition, la mise en forme de texte est moins agréable qu’avec Microsoft Word car il faut l’écrire au lieu de sélectionner une mise en forme à la souris ou par raccoucis clavier (Ctrl+I italique ; Ctrl+G ou Ctrl+B gras ; et Ctrl+U souligné).
    Sublime Text permet de faire la même chose :

    1. sélectionner le mot/la phrase (double-clic ou à la souris)

    2. taper _ → Sublime Text entoure le mot/la phrase par des "tirets bas" :
      _italique_ → italique

    3. taper * → Sublime Text entoure le mot/la phrase par des "étoiles" :
      *étoile* → étoile

  • lors de l’édition : Sublime Text conserve le principe des caractères englobant . avant d’écrire un mot en italique, il suffit de taper le caractère de mise en forme requis et Sublime Text va automatiquement en écrire 2, puis placer le curseur au centre…​ il suffit d’écrire à l’intérieur de ces caractères de lise en forme pour l’obtenir à la volée (ce texte a été mis en forme à la volée…​ garanti par l’auteur!)

Correction orthographique en Français

  • Installation du dictionnaire

    • Maj+Ctrl+P puis install package

    • tapez french & sélectionnez Langage French puis Entrée

  • Activation de la correction orthographique

    • View  Dictionnary  Langage - French  fr-FR

    • View  Spell Check ou le raccourci F6

En cas de problèmes, vous pouvez procéder à l’installation manuelle ci-dessous

  • téléchargez les fichiers de correction orthographique pour le Français directement :

    • French.txt

    • French.dic

    • French.aff

  • enregistrez-les dans : C:\Users\olivier\AppData\Roaming\Sublime Text 3\Packages\User\Dict-FR C:\Users\olivier\AppData\Roaming\Sublime Text 3\Packages\User\Dict-FR[Dict-FR] (à créer)

  • éditez les préférences de Sublime Text avec Preferences  Settings

    • une autre fenêtre de Sublime Text s’ouvre, c’est le panneau de droite qu’il faut éditer (preferences.sublime.user.settings)

Ajout de plugins pour Asciidoc

  • Ctrl+Maj+P ouvre la "pallette" de commande

  • tapez install package

  • tapez (dans la nouvelle fenêtre) asciido et Sublime Text vous proposera d’installer tout ce qui est dispo pour le format Asciidoc

Sélection multiple

  • sélectionner un fragment de texte/mot

  • Ctrl+D successif sélectionnera successivement les instances suivantes du même mot, qu’il sera possible d’éditer en même temps…​

Sélection en mode colonne

  • ClicDroit+Maj

Recherche mot sur plusieurs fichiers

  • Ctrl+Maj+F : un bandeau bas apparaît avec le texte à chercher

  • Entrée : cela cherchera le texte dans tous les fichiers du répertoire (on peut aussi configurer l’étendue de la recherche en cliquant sur la barre médiane)

  • Sublime Text listera les résultats dans un nouveau fichier

GoTo Anything

  • Ctrl+P puis la chaîne de texte : Sublime Text trouvera de suite dans quel fichier cette chaîne de texte est située!

Themes

  • Ctrl+Maj+P (command palette)

  • tapez Themes et choisissez celui qui vous est proposé dans la liste

  • vous pouvez en installer d’autres :

    • Ctrl+Maj+P + install package

    • tapez theme et choisissez un (essai avec Elementary, puis Entrée

    • Ctrl+Maj+P + theme → choix Elementary

Sublime Merge

  • Client Git qui affiche les fichiers modifiés par rapport au dépôt distant

Projets

  • Si vous ouvrez un dossier (ou plusieurs dossiers) à l’intérieur de Sublime Text, vous pouvez enregistrer cette collection de dossiers en tant que projet.

  • à l’aide du raccourci Ctrl+P, une fenêtre de tous les fichiers contenus dans ce dossier s’affichera, et il vous suffira de le sélectionner dans la liste déroulante, sans avoir à naviguer dans un explorateur de fichier.

  • NB si vous avez affiché la Side Bar (soit par View  Side Bar  Shox Side Bar ou par Ctrl+B), vous pouvez aussi cliquer sur un fichier dans la liste …​

  • vous pouvez ajouter des dossiers à ce projet (donc vus dans la "Side Bar"), même s’ils ne sont pas enregistrés au même endroit. Très pratique pour utiliser des fichiers comme exemple sans les copier dans le dossier de travail …​

    • Ajout d’un dossier au projet en cours : Project  Add Folder to Project

    • retrait d’un dossier au projet en cours : Project  Remove Folder from Project : Attention! cette commande est vraiment différente de supprimer le dossier…​ qui le…​ supprimera du disque dur!
