Correction aorthogaphiique (← oups) avec ATOM

Constat d’absence d’un dictionnaire français disponible "globalement" sur Windows 10

Bien que l’on ai francisé l’interface, le correcteur orthographique d’Atom ne fonctione pas, un message d’alerte nous le signale à chaque ouverture du logiciel.
Comme ce problème a été récurrent avec la configuration de Sublime Text et Visual Studio Code, il est temps de trouver une méthode plus globale

Méthode d’installation globale de dictionnaire français

  1. télécharger les dictionnaires français sur Grammalecte

  2. décompresser l’archive hunspell-french-dictionaries-v6.4.1.zip dans C:/DictionnaireFR (répertoire à créér)

  3. Indiquer à Atom l’emplacement de ces fichiers :

    • Fichier  Préférences cliquer sur l’onglet Packages à gauche

    • Descendre jusqu’au paquet spell-check et cliquer sur Settings

    • cocher (si ce n’est pas fait) Use Locales et indiquer fr-FR puis indiquer l’emplacement des dictionnaires C/DictionnaireFR

    • redémarrer Atom

  4. cette méthode devrait porter ses fruits pour Sublime Text et Visual Studio Code …​

la correction orthographique logicielle reste inexacte dans tous les softs d’édition…​
Ici, dans Atom, on a le strict minimum : pas de correction grammaticale, ni othographique pour les titres ou les termes de spécifiques asciidoc…​

AsciidocPreview

Le paramétrage additionnel du paquet AsiidocPreview améliore l’export en .html et en .pdf à partir du panneau de prévisualisation, en effectuant un clic-droit dessus. Les étapes nécessaires sont peu nombreuses :

Export en html :

  • Fichier  Préférences  Packages

  • Choix : asciidoc-preview → Paramètres

    • Synchronise preview pane

    • Show title

    • Always enable Asciidoctor experimental extensions
      (pour avoir le rendu de ces macros asciidoc :)
      Fichier  Ouvrir → Fichier  Ouvrir
      Ctrl+Alt+Suppr → Ctrl+Alt+Suppr
      F13 → F13

    • Always enable compatibility mode

    • Safe Mode : sélectionner le mode unsafe

    • Save as HTML → Open the generated HTML file in your default browser

      Export en pdf :

    • Enable PDF export using Asciidoctor PDF
      WARNING: il faut avoir installé au préalable asciidoctor-pdf (voir la page dédiée : Installation de asciidoctor-pdf)

    • _Automatically opne the PDF after the generation with the default PDF viewer

En images :

Utilisation courante

  1. Édition d’un document texte au format asciidoc :

    • enregistré avec l’extension .adoc

    • utilisant la syntaxe idoine, gras pour gras, italique pour italique, etc.

  2. Prévisualisation sur le panneau de droite, en le demandant à Atom :

    • soit Packages  AsciiDoc Preview  Toggle soit Ctrl+Maj+A

  3. Export aux formats .html et .pdf (optionnel avec Antora [1] ) :

    • se positionner sur le panneau de prévisualisation

    • html : Clic droit et Save As  HTML ou Ctrl+Maj+S

    • pdf : Clic droit et Export As  PDF
